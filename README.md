# Grocery Tracker

A simple Flutter grocery tracker app for Android. It lets users add grocery items with name, quantity, price, image, and purchase date. The app saves entries locally and displays spending analytics.

## Setup

1. Install Flutter from https://flutter.dev/docs/get-started/install
2. Open this folder in VS Code
3. Run `flutter pub get`
4. Run the app on an emulator or device:
   - `flutter run`
5. To build an APK:
   - `flutter build apk`
   - The APK will be available under `build/app/outputs/flutter-apk/app-release.apk`

## Notes

- If you do not yet have an Android project structure, run `flutter create .` in this folder after installing Flutter.
- The app stores grocery data in a local JSON file.
